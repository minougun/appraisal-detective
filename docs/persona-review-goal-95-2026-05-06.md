# 鑑定探偵 ペルソナレビュー95点ゴール

## ゴール

5名の主要ペルソナ評価の平均を95点以上にする。実ユーザー調査ではなく、ローカル完成判定用の設計レビュー基準として扱う。

## ペルソナ

1. P1 ADV・捜査読み好き
   - 会話の緊張、証拠提示、誤提示への反応、捜査ADVとしての読み心地を見る。
2. P2 週1カジュアル
   - 15分前後で迷わず終われるか、各画面の終了条件が見えるか、初回負荷が低いかを見る。
3. P3 鑑定プロシマ
   - 用語と状況の整合、誤選択フィードバック、教材・復習ツールとしての信頼性を見る。
4. P4 感覚過敏・配慮
   - reduced motion、低刺激モード、SE制御、色だけに頼らない情報提示を見る。
5. P5 リプレイ・スコア厨
   - 監査レビュー、通常レビューの次周チェック、記録、ランク、最適化余地を見る。

## 採点基準

各ペルソナ20点満点、合計100点。

- 4点: 初回5秒のフック
- 4点: 迷わない操作導線
- 4点: ゲームらしい手触りと報酬演出
- 4点: 職業追体験・専門性
- 4点: リプレイ動機

平均95点以上の判定には、以下の実装証拠を必須にする。

- 3案件すべてに生成PNGの現地調査ビジュアルがある。
- 案件カードにも現地画像が表示され、初回画面が文字だけにならない。
- 依頼者3名と先輩鑑定士のビジュアルがある。
- 各フェーズ冒頭がノベルADV風シーンとして進み、背景、立ち絵、台詞送りで事件の文脈が伝わる。
- 証拠提示に提示順があり、報告が3枚の論証として見える。
- 結果画面にランクスタンプがある。
- 通常/監査レビュー、監査失敗パターン、SEトグル、生成PNG配信を自動テストする。
- 低刺激モードで圧力フラッシュを無効化できる。
- 各フェーズに終了条件が常時表示される。
- 誤選択時に学びカードが返る。
- 通常レビューでも次周チェックリストが表示される。
- 調整幅は根拠カード2枚で支える必要があり、価格判断と証拠収集が接続している。
- 再反論は提示済み証拠カードで支える必要があり、報告対決が一往復で終わらない。
- 周回ごとに市場シナリオが変わり、同じ案件でも重視すべき証拠が揺れる。
- フォントCDNに依存しない。
- ペルソナ採点チェックが平均95以上で通る。
