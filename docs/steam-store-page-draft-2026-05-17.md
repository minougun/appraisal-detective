# Steamストアページ草案 2026-05-17

## 公開名

鑑定DE探偵

## 英語名

Appraisal Detective

## 短い説明

依頼者の希望に流されず、現地調査・資料照合・3枚の根拠カードで評価額を説明する、不動産鑑定評価ミステリー。

## Short Description

A real-estate appraisal mystery game where you investigate properties, compare documents, and answer client pressure with three evidence cards.

## 長い説明

`鑑定DE探偵` は、不動産鑑定士として依頼者の都合と評価根拠を切り分ける、短編ケース型の鑑定評価ゲームです。

プレイヤーは事件ファイルを選び、現地確認で違和感を拾い、登記・取引事例・レントロール・都市計画などの資料を照合します。最後は3枚の根拠カードを選び、依頼者からの最後の圧力に対して、評価額を説明します。

相続協議、収益物件、再開発予定地、担保評価、借地権、底地、区分所有、ホテル、物流倉庫、海外案件まで、10件のケースを収録しています。

## Features

- 10件の不動産鑑定ケース。
- 現地確認、資料照合、評価判断、報告対決の5フェーズ構成。
- 依頼者の希望に対して、根拠カードで線を引く報告対決。
- 通常レビューと監査レビュー。
- 市場シナリオによるリプレイ変化。
- キーボード、マウス、ゲームパッド操作。
- 低刺激モード、効果音/BGM/VOICE切替。
- 実行中の生成AI処理なし。画像素材は事前生成アセット。
